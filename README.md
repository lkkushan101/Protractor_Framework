# Protractor_Framework
Following should be installed 
1. NodeJS - https://nodejs.org/dist/v14.15.1/node-v14.15.1-x64.msi
2. Protractor - npm install -g protractor
3. WebDriver Manager - webdriver-manager update and then webdriver-manager start
4. Jasmin 2 HTML reporter - npm i protractor-jasmine2-html-reporter

Features 
1. Page Object Model
2. JSON data driven 
3. Jasmin 2HTML reporter
